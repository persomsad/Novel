## ADR 状态说明

- **draft**：草稿状态，决策尚未最终确定
- **active**：已生效，当前正在使用
- **deprecated**：已废弃，被新的 ADR 替代
- **superseded**：被其他 ADR 替代（会注明替代者）

---

## 如何添加新的 ADR

1. 复制 ADR 模板（参考 ADR-001）
2. 编号递增（ADR-002, ADR-003...）
3. 填写决策内容（背景、决策、备选方案、后果）
4. 更新本索引文件
5. 在相关 Issue/PR 中引用 ADR

---

## 决策心跳

- **每个 Milestone 启动时**：回顾 ADR 清单，检查是否需要新增决策
- **每个 Milestone 收尾时**：确认所有 ADR 仍然有效
- **遇到 P0/P1 事故时**：立即补齐相关 ADR
